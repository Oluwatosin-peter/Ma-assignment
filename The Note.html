WHAT IS A WEB

  • THE INTERNET :

    • The internet is a massive network of networks. It contains millions of computers together globally forming a network in which any computer can communicate with another computer as long as they are both connected to the internet.

  • THE WORLD WIDE WEB :

    • The world wide web is a way of accessing information over the medium of the internet. It is an information sharing model that is built on top of the internet {A bunch of connected computer that have website}

  • A BROWSER :

    • A browser is a program that allows you to interact and view contents of the web application programs dedicated for web browsing eg Opera, Firefox, Chrome, Internet explorer e.t.c are called browsers.

  • CLIENTS:

    • Clients are devices {e.g Laptops, Tablet, Mobile, Phone e.t.c} from which browsers are used are referred to client

  • SERVERS :

    • When a computer is connected to the internet and spinning out a website, we call the computer a server because it is serving the website.
